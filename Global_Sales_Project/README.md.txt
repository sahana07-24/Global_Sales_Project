# Global Sales Performance Analysis

## Objective
Analyze global sales data to uncover business trends, identify high-performing markets and categories, and provide actionable recommendations.

## Dataset
51,290 records, 30 features after cleaning and feature engineering.

## Tools & Skills
- Excel
- Python (Pandas, NumPy, Matplotlib)
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization

## Key Insights
- APAC is the highest revenue market
- Technology is the most profitable category
- 2014 recorded the highest annual sales
